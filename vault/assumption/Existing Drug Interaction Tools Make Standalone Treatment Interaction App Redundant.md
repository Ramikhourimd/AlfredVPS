# Existing Drug Interaction Tools Make Standalone Treatment Interaction App Redundant

## Claim

Tools like Lexicomp, Epocrates, and UpToDate already provide drug-drug interaction checking. In psychiatry, the primary clinical goal is to avoid polypharmacy interactions altogether, making it unclear what problem a new standalone treatment interaction application would solve beyond existing tools.

## Basis

An unnamed person approached Rami about building a treatment interaction application (first mentioned 2025-11-13, discussed more explicitly 2025-12-02). Rami expressed skepticism from his clinical perspective: the goal with most patients is to avoid interactions entirely, and existing interaction-checker tools already serve that function.

## Evidence Trail

- 2025-11-13: Early fragmentary discussion — active review of an application concept, phone notification routing discussed
- 2025-12-02: More explicit conversation — Rami expressed skepticism, no commitments made, concept effectively shelved

## Impact

This assumption led to the treatment interaction application concept being deprioritized. If a compelling differentiator emerges (e.g., psychiatry-specific interaction patterns not covered by general tools, integration with the Telia'z clinical platform), this assumption could be challenged.

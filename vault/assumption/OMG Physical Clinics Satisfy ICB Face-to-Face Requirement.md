# OMG Physical Clinics Satisfy ICB Face-to-Face Requirement

## Claim

OMG Healthcare's physical clinic infrastructure in Leeds and across England satisfies ICB commissioners' preference for face-to-face clinical services. By partnering with OMG, Telia'z can offer a hybrid model — OMG provides physical premises for in-person triage and initial consultations, while Telia'z delivers the psychiatric assessment virtually — thus meeting the face-to-face criterion in NHS procurement bids.

## Basis

OMG has ~100,000 GP patients and ~400,000 acute care patients annually across physical clinic sites. ICB commissioners in Leeds prefer bids offering face-to-face services over purely virtual. The OMG partnership was structured with OMG handling governance, triage, and physical infrastructure while Telia'z provides psychiatric expertise. Sean Rejabris noted the bid must address face-to-face service preference.

## Evidence Trail

- 2025-01-22: GP Confed meeting — ICB face-to-face preference flagged as procurement criterion
- 2025-07-03: OMG pathway design — OMG's physical clinics documented; triage conducted by phone/virtual (not in-person)
- No explicit confirmation that ICB would accept a hybrid model where only triage is face-to-face

## Impact

If this assumption holds, the OMG partnership provides a strategic answer to Telia'z's biggest competitive weakness in NHS procurement. If it fails — if ICB requires the psychiatric assessment itself to be face-to-face — Telia'z's core virtual model would need fundamental restructuring for the NHS pathway.

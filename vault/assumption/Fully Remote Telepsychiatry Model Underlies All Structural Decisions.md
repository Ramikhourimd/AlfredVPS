# Fully Remote Telepsychiatry Model Underlies All Structural Decisions

## Claim

The Telia'z clinic operates on a fully remote telepsychiatry model — all patient care is delivered via telemedicine. This foundational operating paradigm shapes every structural decision in the organizational restructuring: pod coordination is virtual, secretary workflows are digital-first (no physical reception), management oversight is remote, and geographic constraints on hiring are relaxed.

## Basis

The Telia'z Clinic Israel project description states the clinic uses a "Telepsychiatry model (fully remote patient care)" and serves patients "through telemedicine." No source records discuss physical clinic space or in-person operations.

## Evidence Trail

- Clinic Israel project description confirms fully remote model
- Restructuring discussions never mention physical location constraints for staffing
- Secretary role discussions focus on scheduling and digital coordination, not front-desk duties

## Impact

- Pod structure assumes virtual coordination, not co-located teams
- Secretary dual-function assumption (patient-facing vs staff-facing) operates in a digital context
- Management span assumptions (pod leader supervising 10) assume remote oversight
- International expansion replicability assumes the remote model transfers across jurisdictions

# Psychiatrists Diagnose PTSD Based on Context Rather Than Strict DSM Criteria

## Claim

In the Telia'z post-war patient cohort, psychiatrists predominantly diagnosed PTSD despite the available symptom criteria better matching depression. This suggests psychiatrists use contextual knowledge (post-war setting, patient population) to guide diagnoses rather than strictly applying DSM criteria.

## Basis

- AI Expert Agent using pure DSM criteria diagnosed depression more frequently than PTSD
- AI Biased Agent with contextual priming replicated the human psychiatrists' PTSD diagnoses
- Questionnaire data correlates with PHQ (depression) not PCL (PTSD)
- This pattern is consistent with known clinical behavior where psychiatrists consider context beyond checklist criteria

## Evidence Trail

- 2026-02-22: Finding presented by Rami. Dekkel described it as "amazing." Noam noted it requires more detailed data presentation to be convincing in a paper.

## Impact

- Forms the core finding for the proposed Paper 2
- Raises questions about DSM as a diagnostic tool vs. clinical heuristic
- May inform how AI diagnostic systems should handle contextual priming
- Rami suggests surfacing this awareness to Telia'z psychiatrists (not to change practice, but for awareness)

# No Structured Protocol for Case Manager Follow-Up Sessions Beyond Intake

## Constraint

Case managers have a structured intake checklist (developed with Rivi's input) that they find very helpful for first patient interactions. However, no equivalent structured guide exists for follow-up sessions (3rd and 4th visits). Case managers report unclear clinical objectives for these sessions — medication side effects are already covered, but there is no defined protocol for what value to add in subsequent follow-ups.

## Source

Case manager interviews conducted by Rivi Idelman during staff experience research. Case managers explicitly requested a structured follow-up session guide similar to the intake checklist.

## Implications

Without follow-up session structure, case managers may underperform in patient engagement beyond intake. This is both a quality risk (inconsistent follow-up care) and a staff satisfaction issue (unclear role expectations create frustration). It also represents an opportunity to leverage follow-up sessions for patient engagement and data collection.

## Expiry / Review

Active until a follow-up session protocol is developed and deployed.

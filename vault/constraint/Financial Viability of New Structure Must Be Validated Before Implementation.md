# Financial Viability of New Structure Must Be Validated Before Implementation

## Constraint

The restructured organization introduces new management layers (Medical Director, Operations Manager, Administrative Manager per district) and expanded support staff. The financial plan must validate that the clinic can sustain these costs as a standalone unit, with shared services cost allocation clearly defined.

## Source

The task "Review Updated Financial Plan" requires validating financial viability of the restructured organization, "especially given budget constraints discussed in the meeting." This intersects with the existing constraint that budget limits new management hires.

## Implications

- Each new management hire must be justified against clinic revenue
- Shared services cost allocation model between parent company and clinic districts needs definition
- The district model's viability depends on whether a single clinic's revenue can support a full management triad
- UK expansion planning depends on proving the cost model works for the Israel clinic first

## Expiry / Review

This constraint should be reviewed after the financial plan is updated and presented to the board. If the clinic reaches profitability targets, the constraint relaxes for future hires.

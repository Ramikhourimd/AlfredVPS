# Case Manager Role Does Not Include Suicidality Screening

## Context

Following a child intake incident on 2025-11-09 where a child revealed active suicidality (suicide plan and weapon access) that was not apparent from the parent intake, Ahmad (Ahmed Masalha) suggested case managers should screen children for suicidality before psychiatric referral. This raised the question of whether the intake workflow should change.

## Options Considered

1. **Maintain current workflow** — Case managers continue with parent intake only; suicidality assessment remains the psychiatrist's responsibility during psychiatric intake
2. **Add suicidality pre-screening by case managers** — Case managers or team leads would see the child directly to rule out suicidality before psychiatric referral

## Decision

Maintain the current workflow. The case manager's role does NOT include ruling out suicidality. This is not standard practice anywhere in clinical settings. Children may become suicidal at any point, and the psychiatric evaluation is designed to handle this, just like any other psychiatric evaluation.

However, case managers should strengthen their suicidality-related questioning during parent intake (asking about current thoughts, past attempts, self-harm, and death talk from the child).

## Rationale

- Suicidality screening is a clinical psychiatric function, not an administrative intake function
- No other clinical setting requires case managers to rule out suicidality before psychiatric referral
- Parents may significantly underreport suicidal ideation — the psychiatric intake is the definitive assessment
- Adding a screening step would not guarantee detection and would create a false sense of security

## Consequences

- Current workflow continues unchanged for case manager-to-psychiatrist referrals
- Parent intake questioning will be reinforced with explicit suicidality questions
- Anan to explain this to Ahmad to address his concerns

# Migrate Innovation Team From Manus AI to Claude Desktop

## Context

Rivi was trained on Manus AI tools (connectors, skills, Creative Seeds, Gravity) during the 2026-03-17 meeting. Rami is currently setting up team members on Manus AI but indicated a planned migration.

## Options Considered

1. **Stay on Manus AI** — current platform with connectors to Notion, Slack, Supabase, Google Gemini, Perplexity, browser
2. **Migrate to Claude Desktop** — Anthropic's desktop application as the primary AI interface for the team

## Decision

Rami plans to migrate the innovation team from Manus AI to Claude Desktop in the future. Timeline not specified. Current Manus setup serves as interim tooling.

## Rationale

Not explicitly stated in the meeting. Likely driven by Claude's deeper integration capabilities, cost considerations, or Rami's existing Claude Code infrastructure.

## Consequences

- Current Manus-specific skills and connectors will need to be rebuilt or adapted
- Team training on Manus is partially temporary investment
- Migration timing should align with Claude Desktop feature parity for team use cases

# Use Israeli Case Managers as UK Stopgap Before Local Recruitment

## Context

UK clinic launch requires case managers who can work in English. Hiring UK-based case managers before signing the Leon contract and validating operations is premature.

## Options Considered

1. **Recruit UK case managers immediately** — Risks hiring before operations are validated; complex and premature
2. **Use existing Israeli English-speaking case managers temporarily** — Faster, leverages existing trained staff, provides system familiarity
3. **No case managers initially** — Not viable; patients need case management support

## Decision

Use existing Israeli case manager Basem (English-speaking, experienced) as the initial UK case manager. This is explicitly a stopgap — long-term UK case managers will be recruited locally in England.

## Rationale

Rami confirmed Basem has done English-language case management before. This approach: (1) avoids premature UK hiring, (2) ensures the case manager already knows Telia'z systems, (3) provides immediate capacity once Leon contract is signed. Adiel emphasized this is temporary — the long-term model is local UK recruitment.

## Consequences

- Basem's Israeli hours need to be partially redirected to UK
- Basel needs to be informed of the time commitment
- UK case manager recruitment should begin once operations are validated

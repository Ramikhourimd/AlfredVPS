# Implement Automated Patient Reminder Call System

Shira proposed during the 2025-07-14 management meeting that an automated reminder system could eliminate approximately 50% of secretarial workload currently spent on manual patient reminder calls. The system should function like standard HMO (kupat cholim) reminder systems — automated outbound calls or SMS confirming upcoming appointments.

## Context

Currently, secretaries spend roughly half their time calling patients to remind them of scheduled appointments. If a patient doesn't answer, they call repeatedly, consuming significant time. An automated system would free secretary capacity for more complex tasks and reduce dependence on headcount growth.

Shira had discussed initial concepts with [[person/Shachar]] before this meeting, but progress was paused due to Shachar's vacation.

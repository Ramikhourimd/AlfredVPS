# Strengthen Suicidality Questions in Parent Intake Protocol

Reinforce the suicidality screening questions used by case managers during parent/caregiver intake sessions. Ensure all intake staff systematically ask about:

- Whether there have been any suicidal incidents (past statements or behavior)
- Current suicidal thoughts from the child
- Past suicide attempts
- Self-harm behaviors
- Talk about death from the child's side

## Context

On 2025-11-09, a child patient was referred to [[person/Ahmed Masalha]] after a parent intake where parents reported a single suicidal statement months ago with no current concerns. During the psychiatric intake, the child revealed an active suicide plan (wanting to hang himself) and had accessed his father's firearm. This gap between parent reporting and child reality underscores the need to strengthen parent intake questioning.

The senior clinician confirmed the case manager role does NOT include ruling out suicidality — that remains the psychiatrist's domain. However, more thorough questioning during parent intake can provide better context for the receiving psychiatrist.

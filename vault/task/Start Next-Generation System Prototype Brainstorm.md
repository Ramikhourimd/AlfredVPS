# Start Next-Generation System Prototype Brainstorm

Lead a team brainstorming exercise to design the most advanced possible version of the Telia'z clinical platform.

## Context

Rami proposed that the innovation team begin envisioning a next-generation system without constraints — assuming all tools and technologies are available. The output would be a conceptual prototype/vision that informs a yearly development roadmap.

## Approach
1. Survey and map all existing system capabilities across current tools
2. Research what competing/peer clinical platforms offer
3. Brainstorm innovative features (AI, automation, patient experience, clinician experience)
4. Create a conceptual mockup/vision document
5. From the vision, build a prioritized yearly development plan

## Participants
- Rivi (lead researcher and concept designer)
- Ohad Edri (literature and research support)
- Yael Marciano (product perspective)
- Rami (strategic direction and approval)

# Transition Clickx to Individual User Accounts

End shared credential usage in Clickx. Each clinician must have their own account.

## Rationale

- Shared credentials create security and audit risks
- Individual accounts enable proper audit trails
- Accountability is traceable per clinician
- Required for compliance

## Actions Required

1. Audit which clinicians currently share credentials
2. Request individual account creation from Clickx
3. Distribute new credentials to each clinician
4. Deactivate shared accounts
5. Verify all users can log in with individual credentials

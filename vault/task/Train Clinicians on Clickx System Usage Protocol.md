# Train Clinicians on Clickx System Usage Protocol

Ori called about getting clinicians to use Clickx properly during sessions. Key concepts discussed:
- Clickx should be open in the background from the start of the session
- Clinicians should not open it mid-session as it disrupts flow
- Leave a few minutes at the end of the session for data entry
- Avoid overloading the clinical encounter with system requirements

## Context

Shira is well-positioned to guide new clinicians on Clickx as part of their onboarding (Hadrakha). Alternatively, this could be handled through staff training support ("Avir Le-Staff").

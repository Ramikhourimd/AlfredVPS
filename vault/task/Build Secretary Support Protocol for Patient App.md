# Build Secretary Support Protocol for Patient App

## Description

Design a support protocol for clinic secretaries to handle patient complaints related to the HealthyMind patient-facing app. Secretaries will be the first point of contact when patients encounter issues, and they currently have no guidance on how to respond.

## Acceptance Criteria

- [ ] Define common complaint categories (login issues, missing summaries, bugs)
- [ ] Create response scripts for secretaries
- [ ] Establish escalation paths (who to contact for technical issues)
- [ ] Define expected response times
- [ ] Train secretaries on the protocol before app rollout

## Context

Raised by Rami during the 2025-11-10 product meeting. Without this protocol, secretaries will face patient complaints with no guidance.

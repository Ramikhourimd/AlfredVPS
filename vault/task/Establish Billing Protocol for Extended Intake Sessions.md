# Establish Billing Protocol for Extended Intake Sessions

Currently there is no formal protocol for when clinicians conduct extended intake sessions. Noa has been requesting double-intake billing on a case-by-case basis by convincing Rami individually. This is unsustainable and creates inconsistency.

## Context

The need was identified during the 2025-12-04 payroll review. Rami noted this is one reason why the clinic needs a formal protocol framework — specifically for "Sal Shikum" (rehabilitation basket) session billing. The protocol should define:
- When an extended intake is justified
- How it should be logged in the system
- What the payment rate is for extended vs standard intakes

This connects to the broader [[task/Formalize Clinic Protocol Framework Document]] initiative.

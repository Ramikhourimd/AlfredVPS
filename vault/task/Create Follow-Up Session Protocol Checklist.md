# Create Follow-Up Session Protocol Checklist

Create a structured guide for case managers for 3rd and 4th follow-up patient sessions. The intake checklist Rivi built was well-received by case managers who said it clarified expectations. They specifically requested a similar tool for follow-up sessions.

## Context

Case managers report that the first two sessions have clear goals (intake assessment, side effects monitoring), but subsequent follow-up sessions lack defined objectives. Staff improvise content individually. This gap also connects to the pre-treatment patient video — if patients understand follow-up value, they'll be more engaged.

# Define Follow-Up Session Protocol for Case Managers

Create a structured checklist/protocol for 3rd and 4th follow-up sessions between case managers and patients, similar to the successful intake checklist.

## Context

Rivi reported during the 2026-02-18 meeting that she created an intake checklist (with Shira's input) that case managers found very helpful. They requested something similar for follow-up sessions, where:
- 1st and 2nd sessions have clear objectives (baseline assessment, side effects monitoring)
- 3rd and 4th sessions lack defined protocol — case managers improvise and each develops their own approach
- Patients may feel follow-ups are unnecessary, creating tension
- This is an opportunity to define value-add activities for later follow-up sessions

# Define KPI-Based Bonus Structure for Rana

Design a performance-based bonus structure for [[person/Rana Khouri]] to replace the current mekhirot (sales/attendance) incentive. The new structure should:
1. Provide meaningful motivation (not a trivial target)
2. Be tied to measurable performance outcomes (accountability)
3. Be relevant to her actual work scope as Administrative Manager

## Context

The mekhirot incentive was originally introduced when secretaries had patient-sourcing duties. That context no longer applies. Rami directed Basel to take time to understand what metrics make sense before proposing the new structure.

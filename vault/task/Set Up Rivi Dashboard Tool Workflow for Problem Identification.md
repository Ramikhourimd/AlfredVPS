## Task Description

Set up and operationalize the new dashboard tool built for [[person/Rivi Idelman]] that automatically analyzes conversation transcripts to identify operational problems.

## Tool Capabilities

- Analyzes conversation transcripts from clinic operations
- Automatically identifies problems mentioned in conversations
- Assigns responsibility for each identified problem
- Proposes solutions based on the analysis

## Setup Steps

- [ ] Confirm tool is deployed and accessible to Rivi
- [ ] Define the conversation sources to analyze (which channels, meetings, etc.)
- [ ] Establish a review cadence for dashboard findings
- [ ] Define escalation paths for identified problems
- [ ] Train relevant staff on interpreting and acting on dashboard outputs

## Integration

This tool represents a shift from reactive to proactive problem identification. It should integrate with:
- Regular operational review meetings
- The reporting structure defined in [[decision/Empower Renee to Report Operational Issues Directly to Management]]
- The incoming manager's operational oversight workflow

# Phase Out Referral Bonus System by End of December

The current "mekhirot" (referral/sales) bonus system no longer aligns with how the clinic operates. Rami directed that:

1. Communicate to staff that the current system runs through end of December 2025 only
2. Frame it gently — don't take it away abruptly, explain the transition
3. Design a new incentive structure that:
   - Is tied to actual work performance and KPIs
   - Provides genuine motivation (not just an entitlement)
   - Is something the employee can realistically influence
4. Basel to understand KPIs first, estimate amounts, then propose structure
5. Rami will review before anything is communicated externally

## Context

Triggered by [[conversation/Clinic Israel Operations Priorities Discussion 2025-12-02]]. The referral bonus was originally created when there were actual referrals to earn. That model no longer applies.

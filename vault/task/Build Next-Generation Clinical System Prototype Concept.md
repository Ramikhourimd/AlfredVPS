# Build Next-Generation Clinical System Prototype Concept

Survey all existing clinical systems in the market, build a composite model merging the best features, brainstorm innovative/novel additions beyond what exists, and create a visual prototype concept assuming unlimited technical capability. This will serve as the basis for an annual development roadmap.

## Context

Rami proposed this initiative during the 2026-02-18 meeting with Rivi. Rivi should lead starting from existing system analysis. Ohad can support with literature/competitive research.

## Steps

1. Map all existing clinical systems and their capabilities
2. Build composite model combining best features from all
3. Brainstorm innovative additions beyond current market offerings
4. Create visual concept/spec assuming unlimited tools and budget
5. Use prototype to inform annual development roadmap
